# Active task

This file is a free-form scratch slot for whatever task the operator wants the chat agent to focus on.

The refresh script does NOT regenerate this file. Edit it by hand when you want to point a chat agent at a specific concern.

## Suggested format

```
## Goal
<one paragraph>

## Repo(s) involved
<xlotyl, stoneforge, or both>

## Files of interest
- xlotyl/services/...
- stoneforge/packages/...

## Constraints
- preserve <contract>
- run <test> before claiming success
```

## Default state

No active task. See [`../integration/06-current-roadmap.md`](../integration/06-current-roadmap.md) for the next step.
