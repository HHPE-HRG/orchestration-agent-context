## 5. How to find code

1. Open the relevant `02-contract-index.md` for a path-level overview.
2. Open `03-symbol-index.md` (xlotyl or stoneforge) to grep symbol names → file paths.
3. Open the matching `chunks/<topic>.md` to see actual source.
4. Cite both the mirror path (e.g. `xlotyl/chunks/capability-packages.md`) and the canonical path (e.g. `capability-packages/stoneforge-runtime/capability.manifest.json`).

For a more thorough query workflow, see [`HOW_TO_QUERY_CONTEXT.md`](HOW_TO_QUERY_CONTEXT.md).
